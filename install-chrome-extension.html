Chrome浏览器插件安装指南

方法一：ZIP压缩包安装（推荐）

  1. 下载插件ZIP文件：frontend-tools-extension.zip
  2. 打开Chrome浏览器，在地址栏输入：
    chrome://extensions/
  3. 开启右上角的"开发者模式"开关
  4. 点击"加载已解压的扩展程序"按钮
  5. 选择下载的ZIP文件或解压后的文件夹
  6. 插件安装完成！在工具栏可以看到插件图标

方法二：直接加载项目文件夹

  1. 打开Chrome浏览器，在地址栏输入：
    chrome://extensions/
  2. 开启右上角的"开发者模式"开关
  3. 点击"加载已解压的扩展程序"按钮
  4. 选择项目文件夹：
    /Users/lion/Study/common-tools
  5. 插件安装完成！
💡 提示：如果遇到安装问题，请确保Chrome版本在88以上，并且所有文件都在正确的目录结构中。
